# Blingtstic
Blingtastic is an auto voter for contests on [Blingee](http://blingee.com)

## Setup
### Requirements
1. ruby 2.2.2
2. bundler (`gem install bundler`)

### Installation
1. `git clone git@github.com:jayvan/blingtastic.git`
2. `cd blingtastic`
3. `bundle install`
4. `chmod +x voter.rb`

## Usage
`./voter <contest_id> <picture_id>`
